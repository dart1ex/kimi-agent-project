# КОНТРОЛЬ - Платформа мониторинга городской инфраструктуры

## Описание

КОНТРОЛЬ — это независимая платформа для гражданского контроля за состоянием городской инфраструктуры. Пользователи могут сообщать о проблемах через сайт или Telegram бот, а администраторы управляют заявками через админ-панель.

## Функционал

### Для пользователей:
- 📍 Просмотр проблем на интерактивной карте
- 📸 Отправка заявок с фото через сайт
- 💬 Отправка заявок через Telegram бот
- 🏙 Выбор города из списка
- 📋 Лента событий с фильтрацией

### Для администраторов:
- 🔐 Вход в админ-панель с визуальным индикатором
- 📥 "Предложка" — входящие сообщения от пользователей
- ✅ Модерация заявок (одобрить/отклонить/в работу)
- 📊 Статистика по заявкам
- 🔔 Уведомления о новых заявках

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка переменных окружения

Создайте файл `.env`:

```env
BOT_TOKEN=your_telegram_bot_token
ADMIN_ID=your_telegram_id
PORT=3001
```

### 3. Запуск сервера

```bash
npm start
# или для разработки:
npm run dev
```

### 4. Запуск Telegram бота

```bash
pip install aiogram aiohttp
python bot.py
```

## Использование

### Вход в админ-панель

1. Откройте `/admin.html`
2. Введите логин: `admin`
3. Введите пароль: `admin`
4. После входа вы увидите бейдж "Режим администратора"

### Telegram бот

1. Найдите бота `@kontrol_city_bot`
2. Нажмите `/start`
3. Следуйте инструкциям для отправки заявки

### Отправка заявки с сайта

1. Выберите город
2. Нажмите "Сообщить о проблеме"
3. Заполните форму (улица из списка, описание, фото)
4. Отправьте заявку

## API Endpoints

### Публичные

- `GET /api/reports` — Список одобренных заявок
- `GET /api/reports/:id` — Детали заявки
- `POST /api/reports` — Создать заявку (с multipart/form-data)
- `GET /api/streets?city=moscow` — Список улиц города
- `GET /api/bot-info` — Информация о боте

### Админские (требуют авторизацию)

- `POST /api/admin/login` — Вход
- `POST /api/admin/logout` — Выход
- `GET /api/admin/reports` — Все заявки
- `PATCH /api/admin/reports/:id` — Обновить статус
- `DELETE /api/admin/reports/:id` — Удалить заявку
- `GET /api/admin/suggestions` — Предложка
- `PATCH /api/admin/suggestions/:id` — Отметить прочитанным
- `POST /api/admin/suggestions/:id/convert` — В заявку
- `GET /api/admin/stats` — Статистика

## Особенности

### "Предложка" (Inbox)

Все сообщения от пользователей (из Telegram и с сайта) попадают в раздел "Предложка" админ-панели:
- Отображается количество непрочитанных
- Можно отметить как прочитанное
- Можно конвертировать в заявку
- Админ получает уведомление в Telegram

### Выпадающий список улиц

При отправке заявки пользователь выбирает улицу из списка, который зависит от выбранного города:
- Москва: Ленина, Гагарина, Мира, и т.д.
- Санкт-Петербург: Невский проспект, Литейный, и т.д.
- И другие города...

### Загрузка фото

- Поддерживаются форматы: JPG, PNG, WebP
- Максимальный размер: 10 МБ
- Фото отображается в предпросмотре
